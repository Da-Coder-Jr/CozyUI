CozyUI Terminal - Enhance your terminal experience

Usage: cozyui [command] [options]

Core Commands:

theme [name|list] Set or list themes

plugin [action] [name] Manage plugins

config Open configuration

update Check for updates

status Show CozyUI status

help [command] Show help for specific command

Utility Commands:

search Search command history

visual Toggle visual data mode

backup Backup your configuration

restore Restore from backup

Frequently Asked Questions

Will CozyUI slow down my terminal?

Not at all! CozyUI is designed with performance in mind. It's lightweight and optimized to add features without compromising speed.

Is CozyUI compatible with my terminal emulator?

CozyUI works with most popular terminal emulators including iTerm2, Terminal.app, Windows Terminal, GNOME Terminal, Alacritty, and many more.

Can I customize CozyUI to match my workflow?

Absolutely! CozyUI is highly customizable through simple configuration files. You can adjust colors, prompts, shortcuts, and more to suit your preferences.

How often is CozyUI updated?

We release updates monthly with new features and improvements. Security patches are released as needed. The auto-update feature ensures you always have the latest version.
